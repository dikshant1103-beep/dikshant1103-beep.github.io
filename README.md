# dikshant1103-beep.github.io

My portfolio — live at **[dikshant1103-beep.github.io](https://dikshant1103-beep.github.io)**.

> I design machines, then build the physics to prove them.

A single static page, no build step and no framework. HTML, CSS and a little
JavaScript, with video and 3D assets served directly. `index.html` is the
whole site.

## What's on it

**Selected projects** — Chassis Workbench, moto_sim, MambaRUL Studio /
BatteryOS, WarrantyLens, EV Drone Inspector, Career Copilot, RetroFitAI.

**Core mechanical work** — hybrid PSO structural optimization, a digital twin
of a human hand, a stair-climbing mechanism.

**The CAD Vault** — three of my own models rendered in the browser as `.glb`
via `<model-viewer>`: a knuckle joint assembly, a generative motor mount, and
the hand digital twin.

**Experience and education** — vehicle engineering internship, M.Tech in
CAD/CAM. The ATS-formatted résumé is served from the repo root as
`Dikshant_Resume_ATS.pdf`.

## Layout

    index.html              the entire site
    assets/vid/             background and project video
    assets/img/             project screenshots
    assets/models/          .glb models for the CAD Vault
    Dikshant_Resume_ATS.pdf résumé, linked from the page

## Deployment

GitHub Pages, straight off `main` — a push to this repository is a deploy.

and should stay where it is.
